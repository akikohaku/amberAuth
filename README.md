# AmberAuth
一个前端登录系统，这部分是前端代码

包含了许多前端动画与切换逻辑，原程序是为 CSU web 课程开发

支持用户名与邮箱登录，自动识别用户名和邮箱

支持验证码登录，自动识别区分手机号和邮箱

包含了一个简单的后端界面和管理员界面

若要使用验证码，请在后端中配置邮箱账户和腾讯云的APPkey

使用了 css-doodle 动画绘图库

[后端部分 AmberAuth-API](https://github.com/akikohaku/amberAuth-API)
